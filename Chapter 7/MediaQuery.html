Media query is used to create the responsive designs . like @media screen and (max-width: 800px){ } so this defines that this property will be apply to width when it size is from 0 to 800px .


@media only screen and (max-width:500px) { .container{ background-color:black; } }


@media (min-width:484px) and (max-width:811px) { .container{ background-color: aqua; } }


@media (min-width: 828px) and (max-width:1012px) { .container{ background-color: blue; } }
